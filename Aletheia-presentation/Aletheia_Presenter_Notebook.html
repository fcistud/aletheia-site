This week I built an interpretability audit pipeline for a pathology foundation model. I took Phikon-v2 — a vision transformer trained on 460 million histology tiles — and asked: what has this model actually learned? Not what it predicts — what concepts it's formed internally.

I extracted embeddings from 10,000 real colorectal cancer tissue patches, trained a sparse autoencoder to decompose those embeddings into 4,096 individual features, and then tested which features the model actually depends on.

2. Clean biology features

0:25 – 0:40
Here are the clean biology features — high tissue specificity, low correlation with staining properties. Normal colon mucosa — you can see the gland architecture, the crypt structure. Smooth muscle fibres. Lymphocyte infiltration. The model has learned genuine histological concepts.

3. Artifact-sensitive features

0:40 – 0:55
And these are the artifact-sensitive features. Look at the difference. Mixed tissue types — tumor, lymphocytes, mucosa — but what they share isn't morphology. It's staining intensity. They're all strongly stained. The model has learned a second pathway: dark staining equals dense tissue. Within one hospital, that correlation holds. Across hospitals with different staining protocols, it breaks.

Key finding: Biology features alone achieve 97.0% — nearly the full baseline. Remove them and accuracy collapses to 48.1%. Artifact features alone still achieve 91.2% — they carry real tissue information, but through appearance shortcuts that are specific to this staining protocol.
Finding features isn't enough. You have to prove which ones matter. I ran a matched ablation study — 79 clean biology features against 79 artifact-sensitive features.

Using only biology features: 97% accuracy. Nearly identical to the full baseline. These 79 features are essentially sufficient.

Remove them entirely: accuracy collapses to 48%. The model can't function without real morphology.

Now the artifact features. Using only those 79: 91%. Still high. That's the key finding — they carry real tissue information, but through the wrong channel. Through appearance properties specific to this dataset's staining.

The model has two classification pathways. One is robust. One is brittle. Standard metrics can't tell them apart.

5. Representation-level follow-up

1:30 – 1:55 · cut this section if running long
Why this matters: The ablation study operates at the probe level. This follow-up patches features in the SAE's reconstruction of the actual model representation — a step closer to model-level causal intervention. Patching biology features drops accuracy slightly (99.25% → 99.10%). Patching artifact features does not (99.30%). This confirms the shortcut features are part of the learned representation itself.
I also ran two follow-up tests. First, representation-level patching — zeroing features in the SAE's reconstruction of the CLS embedding, not just at the probe level. It confirms the shortcut features are embedded in the model's actual representation.

Second, cross-cluster stability. I clustered patches by staining style and measured how much each feature family shifts. Biology features shift 0.21. Artifact features shift 0.78 — nearly four times as much. The biology is stable across staining conditions. The artifacts are not.
